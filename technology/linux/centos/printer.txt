 Installing HP OfficeJet 4500 printer on Centos 7

 Applications -> Activities Overview -> Settings -> Printers

 Add Printer (su)
 192.168.2.12 -> select: Officejet-4500-G510g-m

  - most default settings
  - output = high quality grey scale

Print test page - !!Success!!
